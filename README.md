# User-Login-Security-Monitor
Program that processes login information from an Excel file and alerts the user of any potential security risks. Risks are scored based on how much of the risk criteria each login attempt checks off. For example, a login attempt made in a suspicious country is given a risk score of 50. The total risk score is evaluated after each column in a row of data is processed and the alert is given a color, for example, red signifies a risk score >70. The entire program is written in Python and the user interface is built using the Streamlit library. Other features include a data frame which previews the data that is being processed from Excel, a slider which changes the speed at which data is processed, and a start/pause feature which allows the user to pause processing to view the current login and the alert history. More information is inlcuded in the writeup.
